Projetos de Gerenciamento de Livros e Editoras

Este repositório contém 4 aplicações web diferentes para gerenciamento de livros e editoras, desenvolvidas com 
tecnologias distintas. Todas as aplicações consomem a mesma API backend e oferecem funcionalidades CRUD completas.

| Projeto | Tecnologia | Framework | Estilo | Ícones |
|---------|------------|-----------|---------|---------|
| livros-servidor | Node.js + Express | Backend API | - | - |
| livros-react | React 19.2.0 | Create React App | Bootstrap | lucide-react |
| livros-angular | Angular 15.2.0 | Angular CLI | Bootstrap | - |
| livros-next | Next.js 16.0.3 | App Router | Bootstrap | - |

**Descrição**: API RESTful desenvolvida com Node.js e Express que fornece endpoints para gerenciamento de livros e editoras.

**Tecnologias**: Node.js, Express, MongoDB (Mongoose), Swagger
